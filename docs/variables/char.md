# **char**

El tipo de dato `char` se utiliza para almacenar un único carácter. Los valores de tipo `char` se escriben entre comillas simples (`'`).

## Ejemplo

```csharp
char letra = 'A';
Console.WriteLine(letra); // Salida: A
```

## Características

- Tamaño: 2 bytes.
- Representa un carácter Unicode.
