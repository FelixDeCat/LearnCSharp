# **bool (Boolean)**

El tipo de dato `bool` se utiliza para almacenar valores booleanos, 
es decir... 
`true` (verdadero) == 1
`false` (falso) == 0

## Ejemplo

```csharp
bool esVerdadero = true;
print(esVerdadero); 
// Salida: true
```

## Características

- Tamaño: 1 byte.
- Valores posibles: `true` o `false`.
