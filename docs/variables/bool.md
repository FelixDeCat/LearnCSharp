# **bool**

El tipo de dato `bool` se utiliza para almacenar valores booleanos, es decir, `true` (verdadero) o `false` (falso).

## Ejemplo

```csharp
bool esVerdadero = true;
Console.WriteLine(esVerdadero); // Salida: True
```

## Características

- Tamaño: 1 byte.
- Valores posibles: `true` o `false`.
